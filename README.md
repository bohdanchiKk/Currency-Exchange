REST API for describing currencies and exchange rates.
Allows you to view and edit lists of currencies and exchange rates, and calculate conversions of any amount from one
currency to another.

## Technologies / tools used:

- JDBC
- Java Servlets
- PostgreSQL
- Docker
- Postman
- Maven
